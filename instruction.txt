Инструкция Git о работе с удаленными репозиториями.
Команда git clone- клонирует переданный репозиторий на ваш компьютер.
Пример: $ git сlone https://github.com/smartiqaorg/geometric_lib.git
Команда git pull- стягивание всех изменений из удаленного репозитория на свой компьютер.
Команда git push- загружает изменения в удаленный репозиторий.
